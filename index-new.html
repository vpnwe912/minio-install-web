MinIO S3 Storage Installation

Get started with MinIO S3 Storage on your Linux server. Choose your preferred installation method below:

Method 1: Using wget

Terminal 
wget https://minio-install.consider.pp.ua/install.sh -O install.sh \
  && chmod +x install.sh \
  && sudo ./install.sh

Method 2: Using curl

Terminal 
curl -fsSL https://minio-install.consider.pp.ua/install.sh | sudo bash
